#  Student Management System (Java)

## Problem
Managing student records manually is difficult and unorganized.

## Solution
This project is a Java-based Student Management System that allows users to add, remove, display, and save student records.

## Features
- Add student
- Remove student
- Display students
- Save to file

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- File Handling

## How to Run
1. Compile all Java files
2. Run Main.java
3. Use menu options
